📖 About This Repository

This repository showcases projects completed as part of the Higher Level Programming curriculum at Holberton School.

SQL: Database creation, queries, and advanced joins

RESTful APIs: API design and integration

Software engineering best practices: Clean code, PEP8/Semistandard compliance, modular architecture, and documentation

Each project adheres strictly to Holberton’s requirements, ensuring executable, fully-documented, and professional-grade code.
