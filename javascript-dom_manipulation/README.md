# JavaScript - DOM manipulation

This project focuses on manipulating the DOM using JavaScript.

## Learning Objectives

- How to select HTML elements in JavaScript
- How to modify element styles and content
- How to handle events
- How to update the DOM without reloading the page

## Requirements

- Code must be semistandard compliant
- No use of `var`
- Files must end with a new line
- All scripts must work in Chrome (version 57.0 or later)
